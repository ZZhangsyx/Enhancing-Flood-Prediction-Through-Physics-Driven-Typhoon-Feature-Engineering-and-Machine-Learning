# Enhancing Flood Prediction Through Physics-Driven Typhoon-Feature Engineering and Machine Learning
This repository includes the Python code for our paper "Enhancing Flood Prediction Through Physics-Driven Typhoon Feature Engineering and Machine Learning". The paper has been accepted by PLOS ONE: 10.1371/journal.pone.0346237.

<img width="1211" height="768" alt="image" src="https://github.com/user-attachments/assets/ec87e58e-2869-417d-8a5f-31b638a82bc6" />

This study focuses on the Boluo watershed, a strategically important sub-basin of the Dongjiang River system within the Pearl River Basin, South China：
<img width="1123" height="715" alt="image" src="https://github.com/user-attachments/assets/72f3887c-2ae7-4f9d-9158-6ed85333e937" />

To validate the necessity and redundancy of the 19 physics-informed typhoon features, a comprehensive sensitivity analysis using RFE and MI was conducted prior to model training and hyperparameter tuning.
