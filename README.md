# Enhancing Flood Prediction Through Physics-Driven Typhoon-Feature Engineering and Machine Learning
This repository includes the Python code for our paper "Enhancing Flood Prediction Through Physics-Driven Typhoon Feature Engineering and Machine Learning".
The code can be fully available without restrictions upon acceptance of the manuscript.
